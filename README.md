# job-tracker-python

## Overview
This project demonstrates database programming using Python and MySQL.
It includes secure database connections and full CRUD operations using
parameterized queries to prevent SQL injection.

## Technologies Used
- Python 3.x
- MySQL
- mysql-connector-python
- Visual Studio Code

## Setup Instructions
1. Install Python 3
2. Install MySQL and MySQL Workbench
3. Clone this repository
4. Install dependencies:
   ```bash
   pip install mysql-connector-python
